# サブシェル

シェルから起動された子プロセスのシェルのこと

シェルの実行の仕方で子プロセスとして実行するか、そのままのプロセスとして実行するかが変わる機能はしらなかった。

そのまま実行パターン
- . ./timer.sh
- source ./timer.sh

子プロセスパターン
- ./timer.sh
- /bin/bash ./timer.sh
